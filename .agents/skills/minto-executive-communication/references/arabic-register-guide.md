# Arabic register guide

Scope: Saudi governmental and regulatory correspondence. Load this guide for
Arabic Tier 2–3 drafts; Tier 1 messages should remain short and natural.

## Honorifics

| Form | Use |
| --- | --- |
| معالي | Use only when present in the thread or supplied by the user. |
| سعادة | Use only when present in the thread or supplied by the user. |
| الأستاذ / الدكتور | Preserve only when already used or explicitly provided. |
| Neutral opening | Use when the correct title is unknown. |

Titles come from the thread or user, never from a role, institution, or guess.

Good: `سعادة المدير، نرفق النسخة المحدّثة.`
Bad: `معالي المدير، نرفق النسخة المحدّثة.` when no title was supplied.

## Openings and closings

Use a neutral opening such as `السلام عليكم ورحمة الله وبركاته،` when the
correspondence does not establish a title. For a known title, use the exact
form already used, followed by a direct first sentence.

Good: `السلام عليكم ورحمة الله وبركاته، نأمل التكرم باعتماد الموعد المقترح.`
Bad: `يسعدنا ويشرّفنا أن نرفع إلى مقامكم الكريم...` when a direct request is due.

Useful closings include `وتفضلوا بقبول خالص التحية والتقدير،` for formal
external correspondence and `شاكرين لكم تعاونكم،` for a concise administrative
close. Match the existing thread when one is established.

## Decisive but deferential verbs

Prefer verbs that identify action and ownership: `نقترح` for a recommendation,
`نطلب اعتماد` for an approval request, `نفيدكم` for a status, and `نلتزم` for
a confirmed commitment. Pair a clear verb with respectful framing rather than
several softeners.

Good: `نقترح اعتماد الخيار الثاني، ونطلب توجيهكم قبل 20 يونيو.`
Bad: `قد نكون ربما نرغب في النظر في إمكانية اعتماد الخيار الثاني.`

## English-to-Arabic pitfalls

Avoid literal idioms, over-flattering openings, mixed registers, and translated
organizational titles. State the administrative meaning in idiomatic Arabic.

Good: `نحتاج إلى موافقتكم قبل يوم الخميس لبدء التنفيذ.`
Bad: `نحتاج إلى موافقتكم في أقرب وقت ممكن` when the actual deadline is Thursday.

Good: `سنوافيكم بالتحديث بعد استكمال المراجعة.`
Bad: `سنرجع إليكم` when the thread requires formal administrative phrasing.

Keep English product or organizational names as supplied unless the user asks
for a confirmed Arabic equivalent.
